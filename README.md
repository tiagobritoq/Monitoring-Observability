# 🚀 Monitoring & Observability Lab

Repositório dedicado a estudos e práticas de **monitoramento, observabilidade e automação**.

## 📂 Estrutura do Projeto

```
Monitoring-Observability/
├── zabbixserver-lab/     # Ambiente Zabbix com Docker (utilizando docker-compose)
├── zabbix-reports/       # Scripts de relatórios via API Zabbix
└── .github/workflows/    # Pipelines CI/CD
```

---

## 🔧 Projetos

### 📊 zabbixserver-lab

Laboratório para provisionamento de ambiente Zabbix.

* Docker
* Simulação de ambiente real
* Monitoramento de hosts e serviços

---

### 📈 zabbix-reports

Scripts para extração de dados da API do Zabbix.

* Relatórios de disponibilidade
* Exportação em Excel
* Automação com Python

---

## 🤖 CI/CD

Este repositório utiliza GitHub Actions para:

* Validar estrutura dos projetos
* Verificar scripts shell
* Garantir consistência dos labs

---

## 🎯 Objetivo

Consolidar conhecimentos em:

* Observabilidade
* Infraestrutura como código
* Automação (DevOps/SRE)
